## VerdantDotFiles

Dotfiles are designed for:
- 1920x1080 screens
- rmpc users
- ColemakDH users
- Green People
- and fans of 初音ミク

## INSTALL


### Stow is used to install
##### !!INSTALL STOW BEFORE PROCEEDING!!
```
sudo pacman -S stow
yay stow
```

# Used Apps:
- Ax-Shell
- cava
- fastfetch
- rmpc

## Cloning and Installing
```
git clone https://github.com/VerdantEli/verdantdotfiles.git ~/.dotfiles
cd ~/.dotfiles
stow *
```
